# Generic installation instructions

```js
import { Penpad, Specimen } from '@penpad/core'
```

Put the `Penpad` component anywhere in your website.

```jsx
<Penpad>
  <Specimen id='SpecimenOne'>
    <button>I am a button!</button>
  </Specimen>

  <Specimen id='SpecimenTwo'>
    <button>I am a button!</button>
  </Specimen>
</Penpad>
```
